# 💰 Tip Calculator

A simple Python program that calculates how much each person should pay when splitting a restaurant bill, including a chosen tip percentage. This beginner-friendly project demonstrates the use of user input, arithmetic operations, variables, and rounding in Python.

---

## 📌 Features

* 💵 Accepts the total bill amount
* 📊 Allows the user to choose a tip percentage (10%, 12%, or 15%)
* 👥 Splits the bill among any number of people
* 🧮 Calculates the total amount, including the tip
* 💲 Displays the amount each person should pay, rounded to two decimal places

---

## 📚 Concepts Practiced

* User input (`input()`)
* Type conversion (`int()`, `float()`)
* Arithmetic operations
* Variables
* Percentage calculations
* Rounding numbers with `round()`
* Formatted string literals (f-strings)
